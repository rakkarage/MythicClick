# 🖱️ MythicClick

![Options Screenshot](screenshot.png)

Awarded teleports highlighted. Left click: Teleport. Right click: LFG.

## Usage

- Open the Mythic+ dungeons panel.
- Left click highlighted dungeon icons to cast the teleport spell (if known).
- Right click dungeon icons to open LFG for that specific dungeon pool.
- No addon settings are required.
